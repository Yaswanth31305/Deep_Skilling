# Student Course Portal — Week 5 DeepSkilling

Student Course Portal application built with Angular v21, RxJS, NgRx state management, Reactive Forms, and REST API integration.

## Key Features
- **Student Dashboard:** Track enrolled courses, GPA, and student profiles.
- **Course Management:** Filter and search courses by category with enrollment actions.
- **Student Registration:** Template-driven registration form with full validation.
- **Course Registration:** Reactive form with dynamic `FormArray` for course preference priorities.
- **State Management:** Centralized state management using NgRx store, effects, and entity adapter.
- **REST API Integration:** Services connected to mock REST backend with HTTP interceptors.

## Quick Start
```bash
# Start Angular dev server
npm start

# Start mock REST API (optional)
npx json-server db.json --port 3000

# Run unit tests
npm test
```
